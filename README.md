# HAKURO Brand Commerce

架空の日本国内向けアパレルブランド「HAKURO / 白露」の EC 兼ブランドサイトです。

## Brand Concept

HAKURO は、都市で働く大人のための「静かな機能服」を提案するプレミアムウェアブランドです。日本の湿度、季節の移ろい、余白の美意識を背景に、上質な天然素材と軽い機能性を日常着へ落とし込みます。

## Target

30代から40代の都市生活者。ロゴや過度な装飾よりも、素材、シルエット、長く着られる品質、信頼できる情報開示を重視する顧客を想定しています。

## Design Direction

余白、タイポグラフィ、写真の間合いで高級感を作ります。2026年の方向性として、ラグジュアリー・ミニマリズム、モバイルファーストの商品導線、透明性のある品質情報、控えめなモーション、質感のあるビジュアル表現を採用しています。

## Run

### Preview without dependencies

この環境では `npm` が見つからない場合があるため、見た目確認用の静的プレビューを用意しています。

```bash
node preview/server.mjs
```

ブラウザで以下を開きます。

```text
http://127.0.0.1:4173
```

スマホなど別端末から確認する場合は、Macと同じWi-Fiに接続し、MacのLAN内IPアドレスを使います。

```text
http://<MacのIPアドレス>:4173
```

例:

```text
http://172.20.10.5:4173
```

`localhost` や `127.0.0.1` は、その端末自身を指すため、スマホでは使えません。

別ポートで起動したい場合:

```bash
PORT=4174 node preview/server.mjs
```

### Next.js development

通常の開発環境では、以下で Next.js 版を起動できます。

```bash
npm install
npm run dev
```

ブラウザで以下を開きます。

```text
http://localhost:3000
```
